# Find the Odd Int



## Instruction

Given an array, find the integer that appears an odd number of times.

There will always be only one integer that appears an odd number of times.



## Test

```
array: [20,1,-1,2,-2,3,3,5,5,1,2,4,20,4,-1,-2,5]
result: 5
```

```
array: [1,1,2,-2,5,2,4,4,-1,-2,5]
result: -1
```

```
array: [20,1,1,2,2,3,3,5,5,4,20,4,5])
result: 5
```

```
array: [10]
result: 10
```

```
array: [1,1,1,1,1,1,10,1,1,1,1]
result: 10
```



## Tag

``ARRAY`` ``SEARCH`` ``INTEGER``
